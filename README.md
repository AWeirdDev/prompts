# prompts
Instructive prompts for everyday use.

See the `prompts/` directory.

Under `awdev/`, the prompts are written by me.

## Commands
```haskell
$ aip add <prompt>...
```

Adds a prompt.


## Syntax
```mdx
---
title: Title
description: Description
author: Author
parameters:
- one
- two
---

Hello, **Markdown!**

- This is optional: {one or default value!}
- This is required: {two}
```
